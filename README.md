># FinCloud
>
>#### 介绍
> 财智云平台是一个基于SpringBoot+Vue的前后端分离微服务项目，使用dubbo+zookeeper进行远程方法调用，同时包含redis缓存、token鉴权、httpclient请求短信和身份证信息验证接口、集成快钱支付、定时任务等功能。可以很好的练习涉及金融投资项目业务的开发。
